# 📖 [MoneyCast] <a name="about-project"></a>

> Description

**MoneyCast** is fullstack mobile web app that helps the user keeps track of his spendings,
it's built using rail a framework built in top of ruby the app is easy to use and has a simple ui and capable of performing all CRUD operations

## 💻 Getting Started <a name="getting-started"></a>

> below is what u can do to get started with this project 


### Prerequisites

In order to run this project you need:

- a recent version of rails installed
- a relational database such as : `postgresql` `sqlit` or `mysql` 
### Setup

Clone this repository to your desired folder:



```sh
  cd my-folder
  git@github.com:ChaabaneAnas/Budget_App.git
```


### Install

Install this project with:



```sh
  cd Budegt_App
  bundle install
```


### Usage

To run the project, execute the following command:



```sh
  rails server
```


### Run tests

To run tests, run the following command:


```sh
  rspec spec
```
